Vår klasse

Hva kan vi gjøre?

1
Snakk åpent om det Start dialogen tidlig. Si åpent om dere er med eller ikke — så vet alle hva man har å jobbe med.
2
Bli enige som klasse Ingen smarttelefon på barneskolen. Enkel telefon for kontakt. Når flertallet holder igjen, forsvinner presset.
3
Gradvis tilgang fra ungdomsskolen Smarttelefon med grenser og åpen samtale — ikke forbud, men dialog.
